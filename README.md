job-to-jorb
=============

![](logo.png)

Chrome extension that replaces occurrences of 'job' with 'jorb' or its many variants.

[Direct download of crx file](https://github.com/tanmaniac/job-to-jorb/blob/master/JobtoJorb.crx?raw=true)
